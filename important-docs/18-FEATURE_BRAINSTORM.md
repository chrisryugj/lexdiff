# 공무원 대상 신규 기능 브레인스토밍

**작성일**: 2026-03-15
**타겟 사용자**: 지자체 공무원, 국가 공무원 (법무담당관, 민원담당, 인허가담당, 정책담당)

---

## 배경

LexDiff는 현재 7대 핵심 기능(AI 자연어 검색, AI 요약, 신구법 대비표, 3단 비교, 위임법령 추적, 영향 추적기, 통합 판례 검색)을 보유하고 있으며, FC-RAG 엔진 + korean-law-mcp 도구 기반으로 법제처 전체 데이터에 실시간 접근 가능하다.

아래는 이 인프라 위에 구축 가능한 공무원 업무 혁신 기능 후보들이다.

---

## Tier S — 게임체인저급 (공무원 업무 혁신)

### 1. 기안문 자동 생성기 (Draft Generator)

**개요**: AI가 법령 조문 + 판례 + 해석례를 근거로 공문서 기안문 초안 자동 생성

**사용 시나리오**:
```
입력: "건축법 제11조 위반 건축물에 대한 시정명령 기안문 써줘"
출력:
  - 법적 근거 (건축법 제79조, 시행령 제115조) 자동 삽입
  - 처분 사유 기재
  - 이행기한 포함
  - 공문서 양식(두문-본문-결문) 자동 포맷팅
```

**세부 기능**:
- 처분서, 통보서, 회신서, 질의서 등 유형별 템플릿
- 법적 근거 조문 번호·항·호 정확 인용
- 기존 해석례/판례에서 유사 사례 자동 참조
- 기안문 법적 리스크 자동 검토 (근거 법령 유효성 확인)

**기술 구현**:
- FC-RAG 엔진 위에 출력 포맷 레이어 추가
- 공문서 양식 템플릿 시스템
- 인용 검증 시스템 재활용

**타겟**: 지자체 민원 담당자, 인허가 담당자, 법무 담당

**구현 난이도**: ★★☆☆☆ (기존 인프라 최대 활용)

### 2. 자치법규 충돌 검사기 (Ordinance Conflict Detector)

**개요**: 지자체 조례가 상위법(법률/시행령)과 충돌하는지 자동 탐지

**사용 시나리오**:
```
입력: "서울시 건축조례 제5조가 건축법 시행령과 충돌하는지 확인해줘"
출력:
  - 건축법 시행령 §61 위임범위 분석
  - 조례 조항별 적법성 판단
  - 위임 범위 초과 여부
  - 법률 개정 후 미반영 조항 식별
```

**세부 기능**:
- 조례 전문 입력 → 조항별 상위법 근거 자동 매칭
- 위임 범위 초과(ultra vires) 자동 판단
- 상위법 개정 후 미반영 조항 식별
- 다른 지자체 동일 조례와 비교 분석
- 충돌 심각도 분류 (위법/위임초과/미반영/권고)

**기술 구현**:
- 기존 조례 검색 API (`ordin-search`, `ordin`) + 3단비교 + 영향추적기 조합
- 조례↔상위법 매핑 로직 신규 개발
- AI 판단 레이어 (위임범위 분석)

**타겟**: 지자체 법무담당관, 자치법규심의위원회

**구현 난이도**: ★★★☆☆

### 3. 법령 개정 영향 시뮬레이터 (Amendment Impact Simulator)

**개요**: 조문 수정안 입력 → 영향받는 하위법령/조례/판례 목록 자동 산출 (개정 "전" 예측)

**사용 시나리오**:
```
입력: "건축법 제11조에 '사전환경영향평가' 요건을 추가하면 영향 범위는?"
출력:
  - 영향받는 시행령 조항 목록
  - 개정 필요한 지자체 조례 수 (전국)
  - 관련 판례에서의 기존 해석 변화 예측
  - 개정 우선순위 로드맵
```

**세부 기능**:
- 현재 영향 추적기의 **역방향 확장** — 사후 분석 → 사전 예측
- 파급 범위 시각화 (트리 그래프)
- 입법영향분석서 초안 자동 생성
- 개정 일정 시뮬레이션

**기술 구현**:
- 영향추적기(`impact-tracker`) 코어 로직 재활용
- 역방향 탐색 알고리즘 추가
- 조례 데이터 전국 스캐닝

**타겟**: 법제처, 국회 입법조사처, 각 부처 법무 담당

**구현 난이도**: ★★★★☆

### 4. 민원 자동 회신 보조 (Civil Complaint Response Assistant)

**개요**: 민원 내용 입력 → 관련 법령/판례/해석례 기반 회신 초안 생성

**사용 시나리오**:
```
입력: "음식점 옆에 축사를 지어도 되는지 문의하는 민원에 답변해줘"
출력:
  - 관련 법령: 축산법 §3, 건축법 §44, 해당 지자체 조례
  - 유사 해석례: 법제처 해석 17-0234
  - 관련 판례: 대법원 2023다12345
  - 회신문 초안 (법적 근거 포함)
  - 법적 리스크 경고 (이 답변의 한계점)
```

**세부 기능**:
- 유사 기존 해석례 자동 매칭
- 법적 근거 신뢰도 표시 (법률 > 시행령 > 해석례 > 판례)
- 답변의 위법 소지 자동 감지
- 회신 이력 관리 (동일 유형 민원 재활용)

**기술 구현**:
- FC-RAG 엔진 + 해석례 검색 그대로 확장
- 회신문 포맷 템플릿 추가
- 유사 민원 매칭 알고리즘

**타겟**: 모든 민원 담당 공무원 (가장 넓은 사용자층)

**구현 난이도**: ★★☆☆☆ (기존 인프라 최대 활용)

## Tier A — 높은 실용성

### 5. 인허가 체크리스트 생성기 (Permit Checklist Generator)

**개요**: 사업 유형 입력 → 필요한 모든 인허가 목록 + 근거 법령 + 소관 부서 자동 생성

**사용 시나리오**:
```
입력: "태양광 발전소 사업에 필요한 인허가 전부 알려줘"
출력:
  ☐ 전기사업허가 (전기사업법 §7, 소관: 산업통상자원부)
  ☐ 개발행위허가 (국토계획법 §56, 소관: 지자체 도시계획과)
  ☐ 농지전용허가 (농지법 §34, 소관: 지자체 농업정책과)
  ☐ 산지전용허가 (산지관리법 §14, 소관: 산림청/지자체)
  ☐ 환경영향평가 (환경영향평가법 §22, 소관: 환경부)
  → 처리 순서: 환경영향평가 → 농지전용 → 개발행위 → 전기사업
```

**세부 기능**:
- 법령 간 교차 요건 자동 추출
- 처리 순서(선후행 관계) 자동 판별
- 사업 유형별 체크리스트 템플릿
- 지역별 추가 요건(조례) 자동 반영

**타겟**: 인허가 담당, 투자유치 담당, 규제혁신 담당, 원스톱 민원 창구

**구현 난이도**: ★★★☆☆

### 6. 판례 동향 리포트 (Precedent Trend Report)

**개요**: 특정 법령/조문 관련 판례를 시계열 분석하여 트렌드 리포트 생성

**사용 시나리오**:
```
입력: "건축법 위반 과태료 관련 최근 3년 판례 트렌드"
출력:
  - 연도별 판결 건수 추이 (차트)
  - 원고 승률 변화 (2023: 23% → 2025: 41%)
  - 주요 쟁점 변화 (절차적 하자 → 비례원칙 위반으로 이동)
  - 핵심 판결 3건 요약
  - 실무 시사점
```

**세부 기능**:
- 시계열 판례 분석 (연도/분기별)
- 법원 판단 경향 변화 감지
- 행정심판 재결례와 교차 분석
- 자동 브리핑 보고서 생성 (월간/분기)

**타겟**: 법무담당관, 소송 대응 담당

**구현 난이도**: ★★★☆☆

---

### 7. 법령 해석 충돌 탐지기 (Interpretation Conflict Finder)

**개요**: 동일 조문에 대한 복수 해석례/판례 간 모순점 자동 탐지

**사용 시나리오**:
```
입력: "국토계획법 제56조에 대한 해석이 서로 다른 사례 찾아줘"
출력:
  ⚠️ 충돌 발견:
  - 법제처 해석 2021-0456: "~는 허용된다"
  - 대법원 2023다78901: "~는 허용되지 않는다"
  - 행정심판 2024-1234: "조건부 허용"
  → 시간순 해석 변천사 타임라인
  → 현재 유효한 해석 판단
```

**세부 기능**:
- 법제처 해석 vs 법원 판례 vs 행정심판 재결 교차 비교
- 시간순 해석 변천사 타임라인
- AI 기반 모순점 자동 식별
- 현재 유효한 해석 추론

**타겟**: 법무담당관, 정책 담당

**구현 난이도**: ★★★★☆

### 8. 부처 간 소관 법령 매핑 (Cross-Ministry Law Mapper)

**개요**: 법적 이슈 입력 → 소관 부처/과 자동 판별 + 협업 필요 부처 식별

**사용 시나리오**:
```
입력: "노인복지시설에 태양광 패널 설치 관련 법적 검토"
출력:
  주관 부처: 보건복지부 (노인복지법)
  협의 부처:
  - 산업통상자원부 (에너지법, 전기사업법)
  - 국토교통부 (건축법)
  - 소방청 (소방시설법)
  - 지자체 (건축조례, 노인복지조례)
```

**타겟**: 국무조정실, 부처 간 협업 담당, 지자체 기획부서

**구현 난이도**: ★★★☆☆

## Tier B — 차별화 기능

### 9. 조례 벤치마킹 도구 (Ordinance Benchmarking)

**개요**: 동일 주제 조례를 전국 지자체별 비교 분석

**사용 시나리오**:
```
입력: "출산장려금 조례를 전국 광역시 기준으로 비교해줘"
출력:
  | 지자체 | 첫째 | 둘째 | 셋째 | 소득기준 | 거주요건 |
  |--------|------|------|------|----------|----------|
  | 서울   | 100만 | 150만 | 200만 | 없음    | 1년      |
  | 부산   | 50만  | 100만 | 200만 | 중위80% | 6개월    |
  | ...    |      |      |      |          |          |
  → 선진 사례: 세종시 (가장 높은 지원금, 소득기준 없음)
```

**세부 기능**:
- 동일 주제 조례 자동 매칭 (전국 17개 시도 + 226개 기초지자체)
- 규제 강도, 지원 범위, 기준치 비교 표
- 선진 조례 자동 추천
- 조례 제·개정 트렌드 분석

**타겟**: 조례 입안 담당, 정책 연구 담당, 의회 사무처

**구현 난이도**: ★★★☆☆ (조례 검색 API 이미 존재)

### 10. 법령 타임머신 (Law Time Machine)

**개요**: 특정 시점 기준 법령 상태 복원

**사용 시나리오**:
```
입력: "2020년 3월 15일 기준 건축법 제11조 내용 보여줘"
출력:
  - 해당 시점 유효 법령 텍스트
  - 현재 법령과의 차이점 하이라이트
  - 그 사이 개정 이력 (3건)
  - "이 시점 적용 판례" 목록
```

**세부 기능**:
- 이력 기반 특정 일자 법령 텍스트 재구성
- 현재 법령과 자동 diff
- 해당 시점 적용 판례 필터링
- 소급적용/경과규정 자동 안내

**타겟**: 소송 대응 담당, 감사 담당, 과징금 산정 담당

**구현 난이도**: ★★★☆☆ (연혁 API 이미 존재)

### 11. 위임입법 미비 탐지기 (Delegation Gap Detector)

**개요**: "대통령령으로 정한다" → 실제 시행령 존재 여부 자동 크로스체크

**사용 시나리오**:
```
입력: "개인정보보호법에서 위임했는데 하위법령이 아직 없는 조항 찾아줘"
출력:
  ⚠️ 위임입법 미비 발견 (3건):
  1. 제15조제3항 "대통령령으로 정하는 바에 따라" → 시행령 해당 조항 없음
  2. 제28조의7 "총리령으로 정한다" → 시행규칙 미제정
  3. 제39조제2항 → 시행령 존재하나 위임사항 일부 누락
  → 위임 기한 도과 여부: 제28조의7 (시행일로부터 1년 경과)
```

**세부 기능**:
- "~으로 정한다" 패턴 자동 추출 → 하위법령 존재 확인
- 위임 기한 도과 여부 체크
- 부분 미비 (위임사항 일부만 규정) 감지
- 미비 현황 대시보드

**타겟**: 법제처, 국회, 감사원

**구현 난이도**: ★★★☆☆ (3단비교 데이터 재활용)

### 12. 공무원 일일 법령 브리핑 (Daily Law Briefing)

**개요**: 담당 업무 관련 법령 키워드 구독 → 개정/시행/판례 알림

**사용 시나리오**:
```
구독 설정: ["건축법", "국토계획법", "도시개발법"]
매일 아침 브리핑:
  📌 오늘의 법령 변경
  - 건축법 시행규칙 일부개정 (2026.3.15 시행)
    → 변경 요약: 별표1 건축물 용도분류 중 '근린생활시설' 범위 확대
```
