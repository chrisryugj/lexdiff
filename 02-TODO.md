# LexDiff - 남은 작업 목록

## 우선순위: 높음

### 1. AI 검색 결과 내보내기 기능
**설명**: 사용자가 AI 검색 결과를 텍스트 파일로 저장할 수 있는 기능

**구현 내용**:
- AI 답변 섹션에 "내보내기" 버튼 추가
- 다음 정보를 포함한 텍스트 파일 생성:
  - 검색 질문
  - AI 답변 (HTML 태그 제거한 순수 텍스트)
  - 인용된 법령 목록
  - 생성 시간
- 파일명 형식: `AI검색결과_YYYYMMDD_HHMMSS.txt`

**파일 위치**:
- `components/file-search-rag-view.tsx`: 내보내기 버튼 및 로직 추가
- `lib/ai-answer-processor.ts`: HTML → 순수 텍스트 변환 함수 추가

**참고**:
- 기존 복사 기능과 유사하지만 파일로 다운로드
- `navigator.clipboard.writeText()` 대신 `Blob` + `URL.createObjectURL()` 사용
