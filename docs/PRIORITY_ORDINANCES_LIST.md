# 우선순위 조례 목록 (광진구 기준)

## 📋 선정 기준

### 1. 광진구 공무원 실무 중심
- 민원 처리 시 자주 참조하는 조례
- 부서별 업무 수행 필수 조례
- 예산 집행 근거 조례

### 2. 민선8기 공약 연계
- 청년 정책 (일자리, 주거, 창업)
- 돌봄 정책 (아동, 노인, 장애인)
- 안전한 도시 (재난, 범죄예방)
- 친환경 정책 (제로웨이스트, 에너지)
- 지역경제 활성화 (소상공인, 전통시장)

### 3. 타 자치구 비교 수요
- 정책 벤치마킹 대상
- 선진 사례 참조
- 조례 개정 시 참고

### 4. 검색 빈도 및 시의성
- 정책 수립 시 검색 증가
- 예산 편성 시기 검색 증가
- 민원 다발 분야

### 자동 구축 스크립트

**파일**: `scripts/build-ordinance-embeddings.mjs`

```javascript
import priorityOrdinances from '../config/priority-ordinances.json' assert { type: 'json' }

async function buildOrdinanceEmbeddings(district = 'gwangjin') {
  const districtData = priorityOrdinances[district]
  const ordinances = districtData.ordinances

  console.log(`🏙️  Building ${districtData.name} ordinances: ${ordinances.length}`)

  for (const ordinance of ordinances) {
    console.log(`\n📜 [${ordinance.id}/${ordinances.length}] ${ordinance.name}`)
    console.log(`   Department: ${ordinance.department}`)
    console.log(`   Policy Link: ${ordinance.policy_link}`)

    try {
      // 1. 조례 검색
      const searchQuery = `${districtData.name} ${ordinance.name}`
      const searchRes = await fetch(
        `http://localhost:3000/api/ordin-search?query=${encodeURIComponent(searchQuery)}`
      )
      const searchData = await searchRes.json()

      if (!searchData.list?.[0]) {
        console.warn(`   ⚠️  Not found, trying alternative search...`)
        // 대체 검색 로직
        continue
      }

      const ordinSeq = searchData.list[0].ordinSeq

      // 2. 조례 전문 가져오기
      const contentRes = await fetch(
        `http://localhost:3000/api/ordin?ordinSeq=${ordinSeq}`
      )
      const content = await contentRes.json()

      // 3. 임베딩 생성
      await embedOrdinanceArticles(
        ordinSeq,
        districtData.name,
        ordinance.name,
        content.articles
      )

      console.log(`   ✅ Completed: ${content.articles.length} articles`)

    } catch (error) {
      console.error(`   ❌ Error:`, error.message)
    }
  }

  console.log(`\n🎉 ${districtData.name} ordinances completed!`)
}

// 광진구 실행
```

### 조례 활용도 측정

```javascript
// 월별 조례별 검색 횟수
SELECT
  ordinance_name,
  COUNT(*) as search_count,
  AVG(was_helpful) as satisfaction
FROM rag_context_logs
WHERE source_type = 'ordinance'
  AND district = '광진구'
  AND created_at > datetime('now', '-30 days')
GROUP BY ordinance_name
ORDER BY search_count DESC
```
