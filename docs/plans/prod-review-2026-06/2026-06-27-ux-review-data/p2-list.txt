
## 위임법령 three-tier 패널 + 위임입법 미비(gap) 탐지 모달
- [DELEG-3/flow] 미비 탐지 결과 행이 hover 강조는 되지만 클릭/이동이 안 됨 — 해당 조문으로 점프 불가
- [DELEG-4/logic] 위임법령 버튼이 비활성화되면 시행령·시행규칙 없는 조문의 '행정규칙'에 영영 못 들어감
- [DELEG-5/design] 미비 탐지 로딩 중 진행바와 중앙 대형 스피너가 같은 단계를 이중으로 표시

## 법령 비교/참조 모달 (ComparisonModal · ReferenceModal · LawSelectionDialog)
- [F3/design] 조문 미지정(법령 전체) 비교에서 빈 개정이력 사이드바가 기본으로 펼쳐져 폭을 낭비하고 오안내
- [F4/logic] 비교 로드 실패 후 '다시 시도'가 보던 개정버전이 아니라 최신본으로 되돌아간다
- [F5/flow] 참조 모달에서 인용을 여러 단계 타고 들어간 뒤 ESC를 누르면 한 단계가 아니라 전부 닫힌다
- [F6/logic] 법령 선택에서 검색 진행 중 Enter를 누르면 이전 쿼리의 잘못된 1순위가 선택될 수 있다
- [F7/mobile] 모바일 비교 모달의 개정이력 드롭다운이 선택 후에도 활성 버전을 표시하지 않는다

## 조례(자치법규) 벤치마킹 — 전국 지자체 동일주제 조례 검색 + AI 비교 분석
- [OB-4/flow] 지역/전체 필터를 누르면 체크해둔 비교 대상이 전부 초기화
- [OB-5/flow] 긴 결과에서 '비교 분석' 액션이 표 맨 아래 묻혀 체크 후 한참 스크롤
- [OB-6/logic] 지자체 필터 배지가 80개 초과면 통째로 사라짐(설명 없음)
- [OB-7/perf] AI 분석은 취소 불가 + 진행 단계 정보 없는 막연한 스피너
- [OB-8/logic] 지역 선택이 서버 검색을 좁히지 않음 — 후순위 지역은 1000건 캡으로 누락 가능

## 법령 뷰어 (law-viewer): 조문 탐색·사이드바 점프·가상스크롤·위임법령/판례 전환·모바일 바텀시트
- [LV-3/mobile] 모바일 액션버튼 줄이 가로로 잘리는데 스크롤 어포던스가 없어 '위임법령/판례/분석' 발견 불가
- [LV-4/mobile] 모바일 조문목록 바텀시트가 열 때마다 최저 스냅(40vh)으로 리셋 → 매번 끌어올려야 함
- [LV-5/empty-error] 미로드 조문 클릭 시 fetch 중 본문이 '조문을 선택하세요'로 깜빡임 (loadingJo가 본문에 전달되지 않음)

## 판례 (3경로): 직접 검색결과 리스트 → 전문 뷰어 / 법령뷰어 하단 관련판례 → 사이드패널 상세 / 판례 전문 → 관련 심급 네비
- [PREC-6/mobile] 판례 페이지 이동 후 상단으로 스크롤되지 않아 모바일에서 새 결과를 못 봄

- [PREC-8/mobile] 외부출처(국세/지방세) 판례 클릭이 모바일에서 사전 경고 없이 외부 사이트로 튐

- [PREC-10/flow] '판례 (N건)' 버튼은 수백 건을 약속하는데 하단 섹션은 20건만, 더 볼 길 없음
