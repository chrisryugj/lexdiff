# LexDiff 향후 발전 로드맵

**마지막 업데이트**: 2025-11-25

---

## 현재 상태

### 활성화된 시스템
- **Google File Search RAG**: 자연어 AI 검색 (Gemini 2.5 Flash)
- **IndexedDB 캐시**: 법령 내용 7일, 행정규칙 영구
- **Optimistic UI**: 행정규칙 즉시 표시 + 백그라운드 새로고침
- **3단 비교**: 법률 + 시행령 + 시행규칙
- **기본 검색**: law-search API + 레벤슈타인 유사도 매칭

### 비활성화된 시스템
- **Phase 5**: Intelligent Search (학습 데이터 오염)
- **Phase 6**: Vector Search (Phase 5와 함께 비활성화)

### 현재 한계점
1. **오타 처리 제한적**: 일부 패턴만 처리 (벚→법, 셰→세)
2. **동의어 미지원**: "택시" vs "택시운송사업" 연결 불가
3. **맥락 이해 부족**: "수입품에 대한 세금" → "관세법" 직접 연결 불가
   - (File Search RAG로 간접 해결 가능)

## 기능 로드맵

### Phase 8: 스마트 오타 교정 (1주)

**개념**: 사용자 입력의 오타를 자동 감지 및 수정

**구현 방식**: 하이브리드 (규칙 기반 + AI)
```typescript
// 1단계: 규칙 기반 빠른 교정 (~1ms)
// - 자소 분리 오타: "관셰법" → "관세법"
// - 자판 인접 오타: "관새법" → "관세법"

// 2단계: AI 교정 (규칙 실패 시, ~200ms)
// - Gemini Flash로 맥락 기반 교정
```

**UI/UX**:
```
"혹시 '관세법'을 찾으시나요?" [예] [아니오]
```

**예상 효과**: 검색 성공률 +15~20%

**비용**: ~$0.20/월 (100 사용자/일 기준)

---

### Phase 9: 자연어 법령 추천 (2주)

**개념**: 자연어 질문 → 관련 법령 자동 추천

**현재 vs 개선**:
```
현재: File Search RAG로 답변 생성 → 인용 법령 클릭
개선: 검색창에서 바로 법령 추천 → 즉시 이동
```

**예시**:
```
검색: "수입품에 대한 세금은?"
추천: 💡 관세법 (95%) | 대외무역법 (60%)
```

**구현**:
- `/api/analyze-nl-query` 엔드포인트 활용
- 법령 카탈로그 JSON (300+ 법령)
- 사용자 선택 학습

**예상 효과**: 초보 사용자 접근성 대폭 향상

**비용**: ~$0.30/월

---

### Phase 10: 검색 히스토리 & 추천 (1주)

**기능**:
1. **빈도 기반 추천**: 자주 찾는 법령 상위 표시
2. **연관 법령 추천**: "관세법 → 대외무역법"
3. **최근 검색**: 최근 검색어 빠른 접근

**저장 위치**: IndexedDB (브라우저 로컬, 개인정보 보호)

---

### Phase 5v2 & 6v2: 학습 시스템 재설계 (1개월)

**기존 문제**:
- 잘못된 매핑이 학습됨
- 검증 시스템 없음

**해결책**:
```typescript
interface LearningEntry {
  query: string
  lawId: string
  confidence: number       // 0~1
  successRate: number      // 실제 선택 비율
  autoApply: boolean       // 자동 적용 여부
}

// 규칙:
// - successRate < 50%: 학습 데이터 삭제
// - confidence >= 0.8 && successRate > 90%: 자동 적용
```

---

### Phase 11: 법령 연결 그래프 (2-3주)

**개념**: 법령 간 인용 관계 시각화

```
관세법 제38조
  ├─ 관세법 시행령 제45조
  ├─ 관세법 시행규칙 제12조
  │   └─ 관세청 고시 제2024-1호
  └─ 대외무역법 제10조 (관련)
```

**구현**: D3.js 또는 React Flow

---

### Phase 12: 프리미엄 기능 (장기)

1. **AI 법령 해석**: "이 조문을 쉽게 설명해주세요"
2. **법령 변경 알림**: 관심 법령 개정 시 알림
3. **조문 북마크 & 메모**: 개인 메모 + 팀 공유
4. **법령 비교 분석**: 두 법령의 차이점 AI 분석
