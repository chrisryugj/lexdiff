## 핵심 아키텍처 (5단계 검색 전략)
```
L0: 벡터 유사도 (~5ms) → L1: 직접 매핑 (~5ms) → L2: 유사 검색어 (~10ms)
→ L3: 품질 캐시 (~30ms) → L4: API 호출 (500-2000ms) → 자동 학습
```

```bash

cat /home/user/lexdiff/docs/DEPLOYMENT_GUIDE.md  # Phase 1 섹션

# 2. Turso 설정
curl -sSfL https://get.tur.so/install.sh | bash

turso db create lexdiff-feedback
turso db show lexdiff-feedback --url
turso db tokens create lexdiff-feedback

# 3. 환경변수 (.env.local)
TURSO_DATABASE_URL=libsql://...

# 4. 스키마 마이그레이션
```
