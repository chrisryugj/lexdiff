# 우선순위 법령 목록 (Priority Laws List)

## 📋 선정 기준

### 1. 검색 빈도 기준
- 사용자들이 가장 자주 검색하는 법령
- 실무에서 자주 참조되는 법령
- 개정이 빈번하여 비교 수요가 높은 법령

### 2. 국민 생활 밀접성
- 일반 국민의 권리·의무와 직결
- 생활 법률 영역
- 사회적 관심도가 높은 법령

### 3. 기업 실무 중요성
- 기업 컴플라이언스 필수 법령
- 세무·회계 관련 법령
- 수출입·무역 관련 법령

### 4. 기술적 고려사항
- 조문 수가 적당한 법령 (50~500개 조문)
- 구조가 명확한 법령
- API 접근이 원활한 법령

### 자동 구축 스크립트

**파일**: `scripts/build-priority-embeddings.mjs`

```javascript
import priorityLaws from '../config/priority-laws.json' assert { type: 'json' }

async function buildPriorityEmbeddings(phase = 'phase1') {
  const laws = priorityLaws[phase].laws

  console.log(`🚀 Building ${phase}: ${laws.length} laws`)

  for (const law of laws) {
    console.log(`\n📖 [${law.id}/${laws.length}] ${law.name}`)
    console.log(`   Category: ${law.category}`)
    console.log(`   Reason: ${law.reason}`)

    try {
      // 1. 법령 검색
      const searchRes = await fetch(
        `http://localhost:3000/api/law-search?query=${encodeURIComponent(law.name)}`
      )
      const searchData = await searchRes.json()

      if (!searchData.list?.[0]) {
        console.warn(`   ⚠️  Not found, skipping...`)
        continue
      }

      const lawId = searchData.list[0].lawId

      // 2. 전문 가져오기
      const contentRes = await fetch(
        `http://localhost:3000/api/eflaw?MST=${lawId}`
      )
      const content = await contentRes.json()

      // 3. 임베딩 생성
      await embedLawArticles(lawId, law.name, content.articles)

      console.log(`   ✅ Completed: ${content.articles.length} articles`)

    } catch (error) {
      console.error(`   ❌ Error:`, error.message)
    }
  }

  console.log(`\n🎉 ${phase} completed!`)
}

// Phase 1 실행
buildPriorityEmbeddings('phase1')
```
